## maxos 常用记录
### nginx
路径 
```bash
/usr/local/etc/nginx
```
启动命令
```bash
sudo nginx
```
快速停止命令
```bash
sudo nginx -s stop
```
平稳退出命令
```bash
sudo nginx -s quit
```
重新加载配置文件命令
```bash
sudo nginx -s reload
```
